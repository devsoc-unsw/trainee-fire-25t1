## Setup

### Frontend
To, initialise,
```
cd frontend && npm i
```
then run
```
npm run dev
```
to start server

### Backend

To, initialise,
```
cd backend && npm i
```

In `backend/src`, create a .env and include your MONGODB_URI:
```
MONGODB_URI = "insert uri here"
```

then run
```
npm run dev
```
to start server
